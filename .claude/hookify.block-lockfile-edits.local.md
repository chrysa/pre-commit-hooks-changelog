---
name: block-lockfile-edits
enabled: true
event: file
action: block
conditions:
  - field: file_path
    operator: regex_match
    pattern: (package-lock\.json|yarn\.lock|pnpm-lock\.yaml|poetry\.lock|uv\.lock|Cargo\.lock)$
---

🚫 **Blocked: hand-editing a lockfile.**
Lockfiles are generated by their package manager, never edited by hand.
Change the manifest (package.json / pyproject.toml / Cargo.toml) and re-run the
tool to regenerate the lock.
